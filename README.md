# These are my dotfiles.  There may be many like them, but these are mine. 

## Running

```
git clone https://github.com/jonathanpike/dotfiles ~/.dotfiles
cd ~/.dotfiles;
./setup.sh;
```

Setup _should_ be idempotent, but I don't guarantee anything!
